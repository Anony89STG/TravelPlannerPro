## 📋 Descripción

**Travel Planner Pro** es una aplicación web que permite a los usuarios planificar viajes a cualquier país del mundo, centralizando toda la información relevante en un solo lugar. La aplicación consume **4 APIs externas** y almacena datos personalizados usando **LocalStorage**.

### 🎯 ¿Qué problema resuelve?

Normalmente, para planificar un viaje internacional, los usuarios deben consultar múltiples páginas web para obtener:
- Información general del país
- Clima actual
- Tipo de cambio de moneda
- Atracciones turísticas

**Travel Planner Pro** unifica toda esta información en una sola interfaz, ahorrando tiempo y esfuerzo.

## 🛠️ Tecnologías Utilizadas

| Tecnología | Uso |
|------------|-----|
| **HTML5** | Estructura semántica de la página |
| **CSS3** | Estilos, animaciones y diseño responsive |
| **JavaScript (Vanilla)** | Toda la lógica de la aplicación |
| **Fetch API** | Peticiones a APIs externas |
| **Async/Await** | Manejo asíncrono de peticiones |
| **LocalStorage** | Persistencia de datos en el navegador |

> **Nota:** No se utilizaron frameworks como React, Angular o Vue para demostrar dominio de JavaScript puro.
